Domain DB

Database Rules

Data Validation

Basically the data is validated. Not validated information has badge ?.

Owner

Owner data gets from WHOIS. But if owner data is hidden, we use unknown or not validated data (like web information).

Reporter

Reporter is the nickname. There is no validation.

Report Date

Report date is UTC and not validated.

Data Source

We decided on some data sources as the trustable data source. And if data is not from these, put "?" badge into data.

Trustable data source
  • WHOIS
  • Root Zone Database (IANA)
  • Government agencies of member states of the United Nations (list)

Generator Number (in repository)

generator directory has a number that can be prioritized. These are the list of numbers.

  • 01: IANA Information
  • 02: Government agencies Information
  • 80: Third-party Information Source
  • 99: Our Information Source

These rules may change without no announce
